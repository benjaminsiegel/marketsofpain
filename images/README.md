# Images Directory

This folder should contain the following images for the Markets of Pain website:

## Required Images:

1. **poppy-header.jpg** - The poppy plant image with opium latex for the header
2. **author-photo.jpg** - Your professional author photograph
3. **sample-slide.jpg** - The historical Opium Factory plan from Patna

## Instructions:

Save your three provided images to this directory with the filenames listed above. The website is already configured to display them in the appropriate locations.

### Image Specifications:
- **poppy-header.jpg**: Will be displayed as a full-width header banner (recommended: 1920x1080px or larger)
- **author-photo.jpg**: Will be displayed as a circular profile photo (recommended: 500x500px or larger, square aspect ratio)
- **sample-slide.jpg**: Will be displayed as a content image (recommended: maintain original aspect ratio)

All images should be in JPG format for optimal web performance.
